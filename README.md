# 🎓 University AI Chatbot

An advanced AI-powered University Support Chatbot built using Python, Flask, Groq AI, JavaScript, HTML, and CSS.

This project helps students get instant university-related guidance regarding admissions, exams, scholarships, hostel policies, career guidance, and more using conversational AI.

# 🚀 Features

## 🤖 AI-Powered Chatbot
- Groq AI integration using Llama 3.3 model
- Intelligent contextual responses
- Human-like conversations

## 📚 PDF Knowledge Integration
- Reads university PDF documents
- Answers questions from:
  - Admission policies
  - Fee structure
  - Hostel policies
  - Exam rules

## 🎤 Voice Assistant
- Speech-to-text support
- Browser voice recognition
- AI voice response support

## 🧠 Smart AI Personalities
- General Assistant
- Admission Assistant
- Exam Assistant
- Scholarship Assistant
- Career Guide

## 💬 Smart Conversation System
- Follow-up questioning
- Organized responses
- Real-time typing animation
- Session memory

## 📧 Email-Based Session Escalation
- Student enters details before chat
- Chat sessions are emailed to concerned departments
- Auto email sending on page close
- End session functionality

- Responsive design
- Professional chatbot interface
- Smooth animations
- Real-time interaction

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/university-ai-chatbot.git
cd university-ai-chatbot
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
MAIL_USERNAME=your_email@gmail.com
MAIL_PASSWORD=your_gmail_app_password
```

# ▶️ Run Project

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
